# Basic Jurassic Framework Examples

## Setup and Run Server

```bash
# clone project
git clone https://github.com/oknoorap/jurassic.git

# go to this directory
cd examples/basic

# run server
deno run --allow-read --allow-net --allow-env https://deno.land/x/jurassic/server.ts
```
